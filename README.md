An AI-powered web application that helps users prepare for interviews by analyzing resumes and generating personalized interview questions and reports.

🧠 Features
🔐 User Authentication (Register/Login)
📄 Resume Upload & Analysis
🤖 AI-based Interview Question Generation
📊 Detailed Interview Report Generation
📥 Resume PDF Processing

⚡ Fast & Responsive UI
